Neamar

Sous le pseudonyme Neamar, j'ai réalisé une cinquantaine de sites web et d'applications en open-source, en parallèle de ma vie de freelance / développeur / startuper.
Ces projets sont réalisés en Node.js, Flutter, Jekyll, Java ou PHP pour les plus anciens. Vous retrouverez le code source de la plupart d'entre eux sur Github.
